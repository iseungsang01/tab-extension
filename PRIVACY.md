# 개인정보처리방침

Tab Finder for Papers는 사용자가 열어 둔 탭을 빠르게 검색하고, 탭별 메모·사용자 지정 제목·즐겨찾기를 관리하기 위한 Chrome 확장 프로그램입니다.

## 수집 또는 접근하는 정보
확장 프로그램은 기능 제공을 위해 열린 탭의 제목, URL, 도메인, favicon, 활성 상태 등 탭 정보를 읽을 수 있습니다. 또한 검색 정확도를 높이기 위해 열린 웹페이지의 제목 관련 메타데이터(document.title, citation_title, dc.title, og:title, JSON-LD headline, h1 등)를 읽을 수 있습니다.

## 저장하는 정보
사용자가 입력한 탭별 메모, 사용자 지정 제목, 즐겨찾기 상태, 검색 범위 설정은 사용자의 브라우저 로컬 저장소(chrome.storage.local)에 저장됩니다.

## 정보 사용 목적
위 정보는 열린 탭 검색, 자동 제목 표시, 메모 및 즐겨찾기 관리 기능을 제공하기 위해서만 사용됩니다.

## 외부 전송 및 공유
Tab Finder for Papers는 사용자 데이터를 외부 서버로 전송하지 않으며, 제3자에게 판매하거나 공유하지 않습니다.

## 원격 코드
확장 프로그램은 원격 JavaScript 또는 Wasm 코드를 사용하지 않습니다.
